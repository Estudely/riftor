# riftor v3.2.0 — launch

Find the rift. Open it. Cross through.

## Highlights
- **Website:** https://riftor.dev (install, interactive terminal demo, RIFT overview)
- **PyPI project URLs** now point Homepage/Documentation at https://riftor.dev
- **350+** bundled methodology skills (`load_skill` + `/skills`)
- **Baaj / Chakla** subagent dispatch for parallel recon-style work
- Optional **browser_*** tools via `pip install 'riftor[browser]'`
- Scope enforcement, permission gates (diff preview), anti-loop, lessons,
  hypotheses, `/review`, CVSS + md/html/json/SARIF reports

## Install
```bash
pip install -U riftor
# optional SPA recon:
pip install -U 'riftor[browser]'
```
